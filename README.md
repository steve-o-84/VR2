# NEON POUR: Afterdark

**Version 0.6.0**

Ein stilisiertes Anime-Cyberpunk-Barkeeper-Spiel für GitHub Pages, Desktop und den Meta Quest Browser.

## Veröffentlichung auf GitHub Pages

1. Alle Dateien aus diesem Ordner in das Stammverzeichnis deines GitHub-Repositories hochladen.
2. GitHub öffnen: **Settings → Pages**.
3. **Deploy from a branch**, Branch **main**, Ordner **/(root)** auswählen.
4. Die veröffentlichte HTTPS-Adresse im Browser öffnen.
5. Auf dem Quest die Seite direkt im Meta Quest Browser öffnen und **In VR starten** drücken.

## Feste Dateinamen

Die Dateien im ZIP heißen dauerhaft:

- `index.html`
- `styles.css`
- `game.js`
- `README.md`
- `LICENSE`

Nur der ZIP-Dateiname enthält die Version.

## Bedienung

### Desktop

- In der 3D-Bar direkt auf Flaschen klicken, um jeweils 15 ml in den Shaker zu gießen.
- Auf den Eisbehälter klicken, um Eis hinzuzufügen.
- Auf den Shaker klicken, um ihn zu schütteln.
- Auf das Glas klicken, um vom Shaker ins Glas zu gießen.
- Auf die leuchtende **SERVE**-Fläche klicken, um den Drink zu servieren.
- Das Rezept-Tablet rechts zeigt alle Zutaten.

### Meta Quest

- **In VR starten** wählen.
- Die WebXR-Pointer der Controller auf Flaschen, Eisbehälter, Shaker, Glas und Servierfläche richten.
- Trigger drücken, um die Objekte zu bedienen.
