# Velvet Hour Bartender VR — v0.4.0

GitHub-Pages-fertiger WebXR-Prototyp für Meta Quest und Desktop.

## Neu in 0.4.0
- vollständig sichtbares Rezept-Tablet mit drei Cocktails
- Zutaten werden zuerst sichtbar in den Shaker gefüllt
- Flüssigkeitsstand und Mischfarbe verändern sich
- sichtbare Eiswürfel in Shaker und Glas
- Shaker-Schließ- und Schüttelanimation
- sichtbarer Gießstrahl vom Shaker in das gewählte Glas
- Flüssigkeit bleibt im Glas sichtbar
- Desktop-Steuerung und Quest-Lasersteuerung
- Versionsnummer an Startseite und HUD

## Veröffentlichung
Alle Dateien in das Root-Verzeichnis eines GitHub-Repositories hochladen und GitHub Pages für `main / root` aktivieren. Alte Dateien vorher löschen. Danach die Seite mit `?v=040` öffnen, um Browser-Caches zu umgehen.

WebXR benötigt HTTPS und muss im Meta Quest Browser geöffnet werden.
